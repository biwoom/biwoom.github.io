## 왜 "올"인가

경전(經典)의 '經'은 날실을 뜻한다. 부처의 말씀을 실처럼 꿰어 놓았다는 뜻이다. 산스크리트어 'sūtra'도 실, 줄을 의미한다.

"올"은 이 직조의 은유를 이름으로 가져왔다.

```
날실(經) = 경전
씨실(緯) = 논서
올 하나하나 = 개별 개념
천 전체 = 법계(法界)의 총체
```

한 음절, 두 글자 안에 세 개의 층위가 있다.

- **올(실의 올)** — 한 올 한 올 따라가는 깊이. Deep Dive.
- **오르다** — 깊이 파고들수록 높이 올라간다. 상구보리(上求菩提).
- **올(모두)** — 올해, 올것처럼 모든 것을 포괄. 법계의 전체성.

화엄경의 인드라망(因陀羅網)과도 정확히 겹친다. 각 올이 다른 모든 올과 연결되고, 하나를 당기면 전체가 움직인다. OL의 온톨로지 기반 지식그래프가 지향하는 구조와 같다.

## 디자인 시스템 — 눈에 띄지 않는 디자인

OL의 디자인 원칙은 shadcn/ui의 본질과 같다: 사용자가 디자인 자체에 집중하지 않고 콘텐츠에 몰입하는 것.

```
모던   → 무채색 베이스 + 정확한 1px 보더
심플   → 장식 제거, 충분한 여백, 명확한 위계
세련   → 미세한 그림자, 정밀한 간격, 일관된 radius
```

이 원칙은 존재 선언문의 지향과 맞닿는다. OL은 빠른 소비, 과잉 인터랙션, 자극적 UX를 하지 않는다. 깊은 체류와 조용한 탐색을 위해서는 디자인이 투명해야 한다.

### 색상

shadcn의 neutral 팔레트를 채택했다. HSL 기반 토큰으로 라이트/다크 모드를 정의한다. 개인 취향이 강한 컬러는 사용하지 않는다.

### 타이포그래피

장식적 폰트(Playfair Display, Fraunces 등)를 모두 제거하고 시스템 폰트 + Pretendard로 통일했다. 한국어 본문의 가독성이 최우선이다.

### 컴포넌트 매핑

OL의 모든 UI 요소를 shadcn 패턴에 매핑한다. 검색은 Command Palette(Cmd+K), 태그 필터는 Popover + Checkbox, 카드는 Card 컴포넌트. 이 매핑이 일관성을 보장한다.

## 브랜드 보이스

OL이 사용하는 언어의 톤은 세 가지다.

- **명상적** — 서두르지 않고 깊이 생각하게 한다
- **초대적** — 강요하지 않고 탐험을 권유한다
- **지혜로운** — 가르치려 하지 않고 통찰을 나눈다

교조적, 권위적, 상업적인 톤은 피한다. 짧은 문장, 명확한 의미, 은유의 자연스러운 사용. 사용자를 학습자이자 탐험가로 대우한다.

> *올 하나는 작습니다.*
> *하지만 수천 개의 올이 모이면 천이 됩니다.*
> *당신이 선택한 올이 길이 됩니다.*
